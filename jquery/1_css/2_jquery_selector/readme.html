一般选择器

  • id选择器： $("#id")
  • 类选择器： $(".id")
  • 元素选择器： $("div")
  • 全选择器： $("*")
  • 层级选择器(子)： $(".class > p")
  • 层级选择器（后代）： $(".class p")
  • 层级选择器（相邻兄弟）： $(".class + p")
  • 层级选择器(兄弟)： $(".class ~ p")
